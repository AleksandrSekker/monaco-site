# Быстрый старт с Sanity CMS

## Шаг 1: Откройте Sanity Studio

1. Перейдите на https://sanity.io/manage
2. Выберите ваш проект "Monaco Financial Solution"
3. Нажмите "Open Studio" или "Go to Studio"

## Шаг 2: Создайте первый документ

В Studio вы увидите пустой проект. Создайте документы вручную:

### Создание Hero документа:

1. Нажмите "Create" → "New document"
2. Выберите тип документа (если схемы не настроены, создайте простой документ)
3. Или используйте веб-интерфейс для создания документов с полями:
   - `_type`: `hero`
   - `title`: `Одно окно для всех ваших финансовых задач в Монако`
   - `subtitle`: `Банковские счета и карты · Кредитные линии · Инвестиции · Приём и конвертация криптовалюты от 100 000 €`
   - `ctaText`: `Обсудить мою задачу`
   - `guaranteeText`: `100% результат за 90 дней`
   - `guaranteeSubtext`: `или полный возврат гонорара по договору`
   - `stats`: `10+ лет в private banking · Более 400 открытых счетов · Индивидуальные решения для HNWI и компаний`

### Альтернатива: Используйте API напрямую

Если схемы ещё не настроены, можно создать документы через API или использовать простые документы с полями.

## Шаг 3: Проверьте подключение

После создания документов, перезапустите dev-сервер:

```bash
npm run dev
```

Сайт автоматически начнёт получать данные из Sanity (если они есть).

## Важно

Если вы видите ошибки в консоли о том, что документы не найдены - это нормально. Сайт будет работать с текущим статическим контентом до тех пор, пока вы не создадите документы в Sanity.

## Следующий шаг

После создания контента в Sanity, можно интегрировать данные в компоненты сайта. См. инструкции в `SANITY_SETUP.md`.
